# 🍺 Che birra sei?

Un test non scientifico. Da fare con una birra in mano.

**[→ Fai il test](https://TUOUSERNAME.github.io/che-birra-sei/)**

---

Otto domande. Quattro risposte ciascuna. Scopri che birra eri mentre rispondevi.

Domani potresti essere un'altra.

---

## Come funziona

Rispondi alle domande seguendo l'istinto. Se nessuna risposta ti convince, scegli quella che ti fa meno schifo. Alla fine, conta delle lettere rivela il tuo profilo birrario:

- **A** → Heineken (l'adattabile)
- **B** → Tennent's / Viking (l'impulsivo)
- **C** → Corona / BrewDog (il fuggitivo)
- **D** → Bud / Peroni (il pragmatico)
- **E** → Menabrea / Leffe (il misurato)
- **F** → Guinness / Franziskaner (il profondo)

---

## Deploy su GitHub Pages

1. Crea un nuovo repository su GitHub
2. Carica `index.html`
3. Vai in **Settings → Pages**
4. Seleziona branch `main` e cartella `/ (root)`
5. Attendi qualche minuto
6. Il test sarà live su `https://TUOUSERNAME.github.io/che-birra-sei/`
